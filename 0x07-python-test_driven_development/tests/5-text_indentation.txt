====================================

The 5-text_identation take a text and print it with some norme
jump two line when [? : .] encouter

Usage:
=====

importing function from module
    >>> text_indentation = __import__('5-text_indentation').text_indentation

Print a word
    ext_indentation("Hello?")
    Hello?
    <BLANKLINE>

No space are printed in the beginning of the line
    >>> text_indentation("   Hi there.")
    Hi there.
    <BLANKLINE>

Empty arg
    >>> text_indentation("          ")

No space are printed at the end of itch line
    >>> text_indentation("Hello.   ")

    <BLANKLINE>

New line printed only for (? : .)
    >>> text_indentation("No ending period, what bad grammar")
    No ending period, what bad grammar

New lines within a string are printed as normal.
